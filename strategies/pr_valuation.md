# 市赚率估值法 (PR Valuation)

市赚率 (Price-to-Earnings-Return Ratio, PR) 是一种结合了市盈率 (PE)、净资产收益率 (ROE) 和股利支付率的综合估值指标。

## 核心公式

### 第一公式 (标准公式)
适用于 ROE 及分红稳定的价值股。

$$ PR = \frac{PE}{ROE} $$

**修正系数 N (针对股利支付率)**:
$$ PR = N \times \frac{PE}{ROE} $$

其中 N 的取值规则 (以 50% 支付率为标杆):
- **股利支付率 (DPR) ≥ 50%**: N = 1.0
- **股利支付率 (DPR) ≤ 25%**: N = 2.0
- **25% < DPR < 50%**: N = 0.50 / DPR (例如 40% 支付率，N = 1.25)

### 第二公式 (周期股公式)
适用于周期股或困境反转股。

$$ PR = N \times \frac{PB}{ROE^2} $$
*注: 此处 ROE 取景气周期最低值或历史平均值。*

### 第三公式 (指数公式)
适用于不容易查到 ROE 的指数。

$$ PR = \frac{PE^2}{PB} $$

## 估值标准

| PR 值 | 状态 | 操作建议 |
| :--- | :--- | :--- |
| **< 0.6** | 严重低估 | **买入** (4-6折区间) |
| **0.6 - 1.0** | 合理/偏高 | 持有 / 分批卖出 |
| **> 1.0** | 高估 | **卖出** (1 PR 开始卖出) |

*注: 港股建议 0.8 PR 开始卖出; 指数 0.7 PR 可定投，1.0 PR 大幅卖出。*

## 适用场景与限制
1. **适用**: ROE 及分红稳定的价值股 (10% < ROE < 50%)。
2. **不适用**:
   - 成长股 (需要留存利润)。
   - 科技股 (回购代替分红)。
   - 周期股 (需用第二公式)。
   - 银行股 (需考虑拨备率)。
3. **环境调整**:
   - 低利率环境: 合理估值上移至 1.1 ~ 1.4 PR。
   - 高利率环境: 合理估值下移至 0.7 ~ 0.9 PR。

## 计算说明
- **ttmROE**: 最近四个季度 ROE 之和。
- **增长率要求**: 净利润增长率至少达到 ROE 的一半。
