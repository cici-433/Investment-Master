# 国内免费部署方案：本机服务器 + 内网穿透

由于国内云服务器（如阿里云、腾讯云）都需要**付费**且必须进行**ICP备案**才能绑定域名，目前没有像 PythonAnywhere 那样完全免费且省心的托管服务。

对于个人使用的投资系统，**最佳的国内免费方案**是：
**把您自己的 Mac 电脑当作服务器，通过“内网穿透”技术让手机在外也能访问。**

---

## 方案优势
1.  **完全免费**：利用现有电脑和免费穿透工具。
2.  **速度极快**：服务器就在国内（您家里），没有跨国延迟。
3.  **数据安全**：所有数据都在自己电脑上，不用上传到云端。
4.  **无需备案**：不需要购买域名和备案。

## 核心工具：cpolar (推荐)
cpolar 是一款国内流行的内网穿透工具，免费版虽然网址会变化（每次重启后），但对于个人临时查看非常够用。

---

## 操作步骤

### 第一步：在本地启动网站
1.  打开 VS Code 的终端。
2.  运行网站：
    ```bash
    python app.py
    ```
3.  确保看到 `Running on http://0.0.0.0:5000` 字样。

### 第二步：安装 cpolar
1.  访问 [cpolar 官网](https://www.cpolar.com/) 注册一个免费账号。
2.  下载 macOS 版本的客户端。
3.  解压下载的文件，您会得到一个 `cpolar` 可执行文件。
4.  在终端中绑定您的 token（官网后台会提供）：
    ```bash
    ./cpolar authtoken 您的Token字符串
    ```

### 第三步：生成公网链接
在终端中运行以下命令，将本地的 5000 端口暴露到公网：

```bash
./cpolar http 5000
```

### 第四步：手机访问
cpolar 会输出两个链接（http 和 https）。
例如：`https://1a2b3c4d.cpolar.cn`

**直接复制这个链接发到手机微信，打开即可访问您的投资系统！**

---

## 替代方案：Tailscale (更稳定/私密)
如果您希望链接固定，且只有自己能访问（更安全）：
1.  电脑和手机都安装 **Tailscale** APP。
2.  登录同一个账号。
3.  在电脑上启动 `python app.py`。
4.  在手机浏览器输入电脑的 Tailscale IP 地址（如 `http://100.x.y.z:5000`）。
*优点：IP 固定，无需每次重新发链接。*

---

## 总结
*   **如果您想要给朋友看**：使用 **cpolar**。
*   **如果您只给自己看**：推荐使用 **Tailscale**（最稳定）。
*   **如果您希望 24小时在线**：保持 Mac 开机，或者买一个树莓派（~300元）放在家里长期运行。
